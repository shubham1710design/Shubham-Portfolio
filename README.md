# Shubham Portfolio

Personal portfolio website showcasing projects, skills, and contact information.

## Overview

This is a simple, static portfolio site built with HTML, CSS and JavaScript. It showcases projects, a short bio, and contact links.

## Live Preview

Open `index.html` in a browser to view the site locally.

## Features

- Responsive single-page layout
- Project gallery and descriptions
- Contact links and social profiles

## Installation / Local Development

1. Clone or copy the repository to your local machine.
2. Open `index.html` in your browser, or serve the folder with a static server:

## Project Structure

- `index.html` — main HTML file
- `app.js` — JavaScript for interactive behavior
- `style.css` — site styles
- `assests/` — images and other static assets
- `README.md` — project documentation

## How to Update Content

- Edit `index.html` to change structure and content.
- Edit `style.css` to update styles and layout.
- Edit `app.js` to modify interactive behavior.
